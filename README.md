# Business-Analytics-Case-Studies-
# Optimización de Activos: Caso Retail Multi-rubro

Este proyecto utiliza **Python** para resolver un problema estratégico: identificar qué sucursal de una cadena de 4 tiendas es la menos eficiente para proceder con su venta.

## Stack Técnico
* **Lenguaje:** Python
* **Librerías:** Pandas (ETL), Matplotlib/Seaborn (Visualización).
* **Entorno:** Jupyter Notebook / Google Colab.

## Metodología
Se consolidaron 4 fuentes de datos independientes para analizar:
* **Rentabilidad:** Comparativa de facturación bruta vs costos logísticos.
* **Customer Experience:** Análisis de satisfacción basado en ratings.
* **Optimización de Inventario:** Diversificación de categorías por tienda.

## Conclusión Estratégica
Tras el análisis, se recomienda la venta de la **Tienda 4**. Aunque sus costos no son los más altos, presenta el volumen de ventas más bajo y un catálogo desactualizado frente a la competencia de las otras sucursales.
